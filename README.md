# DaYu(大禹)
DaYu(大禹)是一款基于云原生,微服务化的开源数据治理一体化平台，提供一套大型组织的数字化转型平台方案.

DaYu是根据中国夏朝的开国皇帝大禹关于"大禹治水"的事迹命名,面向当前数字化时代,数据洪流进行数字化治理,解决大型企业数字化转型的开源框架.主要包括相关数据治理的理念和工具,通过数据治理,为企业的数字化转型带来商业价值.
